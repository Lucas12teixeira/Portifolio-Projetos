## üé® Screenshot Guidelines

### Tools for Taking Screenshots
- **Windows**: Windows Key + Shift + S, or Snipping Tool
- **macOS**: Cmd + Shift + 4
- **Linux**: Gnome Screenshot, Spectacle, or Flameshot
- **Browser Extensions**: Awesome Screenshot, Nimbus Screenshot

### Best Practices
1. **Resolution**: Use high resolution (1920x1080 or higher)
2. **Format**: PNG for best quality, JPG for smaller file size
3. **Content**: Remove sensitive information (real names, emails, etc.)
4. **Branding**: Ensure consistent styling across screenshots
5. **Focus**: Highlight key features in each screenshot

### Naming Convention
```
feature-name.png
```

Examples:
- `dashboard.png`
- `chat.png`
- `chat-mobile.png`
- `events.png`
- `bible-search.png`
- `profile-edit.png`
